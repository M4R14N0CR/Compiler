Se debe abrir una terminal en la cual nos ubicaremos en la carpeta donde se encuentra el archivo Main.java y ejecutaremos el siguiente comando:

javac Main.java

Luego si no nos da ningun error el compilador ejecutaremos el siguiente comando:

java Main <nombre del archivo de entrada> <nombre del archivo de salida>

en este caso con la direccion donde se ecuentre el archivo de entrada y salida.
como por ejemplo asi:

java Main C:\Users\sjcha\IdeaProjects\miCompilador\src\prueba2.txt C:\Users\sjcha\IdeaProjects\miCompilador\src\salida.txt

Luego navegaremos hasta el archivo de salida y veremos el resultado del analisis lexico.
